Drag this button to your bookmarks bar

🔒  Send Securely
↑ Drag this to your bookmarks bar
Can't drag? Manual install
  1. Right-click the bookmarks bar → Add page / bookmark
  2. Set the name to Send Securely
  3. Paste the code from bookmarklet-code.txt into the URL field
  4. Save
2

Configure on first use

Open OWA, click the bookmark, go to the Settings tab, enter your API token and hit Save. Settings are stored in the browser and persist forever.

3

Use it when composing an email

Open a compose window in OWA, click Send Securely in your bookmarks bar — the panel slides in. Generate the link, then click Replace Email Body.

How it works: The bookmarklet loads a script from joeymeijer.nl/add-in/bookmarklet.js each time you click it, so updates happen automatically without reinstalling. ● No admin needed
